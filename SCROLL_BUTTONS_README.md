# 全局滾動按鈕功能說明

## 功能概述

已成功在系統中添加全局滾動按鈕組件，提供便捷的頁面導航功能。

## 功能特點

### 1. 滾動進度指示器 ⚪
- 顯示當前頁面滾動進度百分比
- 圓形進度環視覺化顯示
- 實時更新滾動位置

### 2. 回到頂部按鈕 🔵
- 點擊快速滾動到頁面頂部
- 平滑滾動動畫效果
- 懸停時有放大動畫

### 3. 跳到底部按鈕 🟢
- 點擊快速滾動到頁面底部
- 平滑滾動動畫效果
- 懸停時有放大動畫

## 顯示邏輯

- 當頁面滾動超過 **300px** 時，按鈕自動顯示
- 在頁面頂部時，按鈕自動隱藏
- 按鈕固定在右下角，不影響頁面內容

## 響應式設計

### 桌面版 (> 1024px)
- 按鈕大小: 48px × 48px
- 位置: 距離右側和底部 16px

### 平板版 (641px - 1024px)
- 按鈕大小: 48px × 48px
- 位置: 距離右側和底部 24px

### 手機版 (< 640px)
- 按鈕大小: 40px × 40px
- 位置: 距離右側和底部 16px
- 圖標大小自動縮小

## 技術實現

### 組件位置
```
src/app/shared/components/scroll-buttons/scroll-buttons.component.ts
```

### 使用的技術
- **Angular 17** Signals API
- **Tailwind CSS** 樣式系統
- **HostListener** 監聽滾動事件
- **SVG** 圖標和進度環

### 集成方式
組件已添加到主應用模板 (`src/app/app.html`)，在所有頁面全局可用。

## 使用方法

無需任何配置，組件會自動在所有頁面上工作：

1. 滾動頁面超過 300px
2. 右下角會出現三個按鈕
3. 點擊相應按鈕進行導航

## 視覺效果

### 按鈕樣式
- **進度指示器**: 白色背景，藍色進度環
- **回到頂部**: 藍色背景，白色圖標
- **跳到底部**: 綠色背景，白色圖標
- **陰影效果**: 懸停時陰影加深
- **動畫效果**: 平滑的過渡和縮放

### 交互反饋
- 懸停時按鈕輕微上移
- 點擊時按鈕回彈
- 背景漸變動畫
- 圖標縮放效果

## 測試方法

1. 啟動開發服務器:
   ```bash
   npm start
   ```

2. 訪問首頁: `http://localhost:4200/home`

3. 滾動頁面測試功能:
   - 向下滾動查看按鈕出現
   - 點擊藍色按鈕回到頂部
   - 點擊綠色按鈕跳到底部
   - 觀察進度指示器的變化

## 自定義配置

如需修改按鈕行為，可以調整以下參數：

### 顯示閾值
在 `scroll-buttons.component.ts` 中修改:
```typescript
this.showScrollButtons.set(scrollTop > 300); // 改為其他數值
```

### 按鈕位置
在組件的 styles 中修改:
```css
.fixed {
  right: 1rem;  /* 調整右側距離 */
  bottom: 1rem; /* 調整底部距離 */
}
```

### 按鈕顏色
在模板中修改 Tailwind 類名:
```html
<!-- 回到頂部按鈕 -->
bg-blue-500 hover:bg-blue-600  <!-- 改為其他顏色 -->

<!-- 跳到底部按鈕 -->
bg-green-500 hover:bg-green-600  <!-- 改為其他顏色 -->
```
