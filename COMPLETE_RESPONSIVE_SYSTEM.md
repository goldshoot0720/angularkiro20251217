## 🏗️ 系統架構

### 核心服務
- **ResponsiveService** - 統一的響應式狀態管理
- 即時監聽螢幕尺寸和方向變化
- RxJS 響應式數據流
- 智能裝置類型檢測

### 響應式組件庫
1. **ResponsiveLayoutComponent** - 主布局容器
2. **ResponsiveContainerComponent** - 智能內容容器
3. **ResponsiveCardComponent** - 自適應卡片
4. **ResponsiveGridComponent** - 智能網格系統
5. **ResponsiveFormComponent** - 響應式表單
6. **ResponsiveNavigationComponent** - 智能導航

### 響應式指令
- **ResponsiveDirective** - 動態樣式控制指令

### CSS 工具系統
- 完整的響應式工具類庫
- 裝置特定顯示控制
- 響應式間距、文字、按鈕
- 自適應圖片和視頻

## 📱 響應式規格

### 斷點系統
- **手機版**: ≤768px - 垂直布局，上側選單
- **平板直向**: 769px-1024px (直向) - 比照手機版
- **平板橫向**: 769px-1024px (橫向) - 比照桌面版  
- **桌面版**: ≥1025px - 水平布局，左側選單

### 智能特性
- **方向感知** - 平板自動根據方向切換布局
- **觸控優化** - 手機版大觸控目標
- **內容適應** - 文字、間距、圖片自動調整
- **動畫過渡** - 平滑的布局切換效果

### 使用示例
```html
<!-- 響應式容器 -->
<app-responsive-container>
  <!-- 響應式卡片 -->
  <app-responsive-card title="標題">
    <!-- 響應式網格 -->
    <app-responsive-grid [mobileColumns]="1" [tabletColumns]="2" [desktopColumns]="3">
      <div>內容 1</div>
      <div>內容 2</div>
      <div>內容 3</div>
    </app-responsive-grid>
  </app-responsive-card>
</app-responsive-container>
```
