# 儀表板功能說明

## 概述
新增的儀表板組件提供了訂閱管理和食品管理的統一視圖，讓您可以快速了解系統的整體狀況。

## 主要功能

### 1. 統計概覽
- **食品總數**: 顯示系統中的食品項目總數和總數量
- **訂閱總數**: 顯示所有訂閱服務的數量和活躍狀態
- **月費總額**: 計算所有訂閱服務的月費總和
- **警告項目**: 統計需要注意的項目（過期食品 + 即將到期項目）

### 2. 時間範圍篩選
- **訂閱管理**: 可選擇 3天 或 7天 的時間範圍
- **食品管理**: 可選擇 7天 或 30天 的時間範圍

### 3. 詳細統計區域

#### 食品管理統計
- 食品狀態分類（正常、即將到期、已過期）
- 即將到期食品列表，包含：
  - 食品圖片預覽
  - 食品名稱和到期日期
  - 剩餘數量和天數

#### 訂閱管理統計
- 訂閱狀態統計（活躍訂閱、即將到期）
- 即將到期訂閱列表，包含：
  - 訂閱名稱和下次付款日期
  - 月費金額
  - 快速訪問網站鏈接

### 4. 快速操作
- 直接跳轉到食品管理頁面
- 直接跳轉到訂閱管理頁面
- 重新載入數據
- 返回首頁

## 訪問方式

1. **側邊欄導航**: 點擊左側導航欄的 "📊 儀表板" 選項
2. **直接訪問**: 瀏覽器地址欄輸入 `http://localhost:4200/dashboard`
3. **首頁快速功能**: 從首頁的快速功能區域點擊 "數據儀表板"

## 技術特點

- **響應式設計**: 支援桌面和移動設備
- **即時數據**: 自動載入最新的食品和訂閱數據
- **智能提醒**: 根據設定的時間範圍顯示即將到期的項目
- **視覺化統計**: 使用顏色編碼和圖標來直觀顯示不同狀態

## 使用建議

1. **定期檢查**: 建議每天查看儀表板，了解即將到期的項目
2. **調整時間範圍**: 根據需要調整篩選時間範圍，獲得更精確的提醒
3. **快速操作**: 利用快速操作按鈕直接跳轉到相關管理頁面進行詳細操作

## 注意事項

- 數據載入可能需要幾秒鐘時間，請耐心等待
- 如果遇到載入問題，可以使用 "重新載入" 按鈕刷新數據
- 確保網路連接正常，以獲得最新的數據更新
