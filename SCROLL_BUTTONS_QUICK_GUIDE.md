# 滾動按鈕快速使用指南

## 🎯 功能說明

已在系統中添加全局滾動按鈕，提供三個便捷功能：

1. **⚪ 滾動進度** - 顯示當前頁面滾動百分比
2. **🔵 回到頂部** - 快速返回頁面頂部
3. **🟢 跳到底部** - 快速跳轉到頁面底部

## 📍 位置

按鈕固定在頁面右下角，當滾動超過 300px 時自動顯示。

## 🚀 測試方法

```bash
# 啟動開發服務器
npm start

# 訪問首頁測試
http://localhost:4200/home
```
